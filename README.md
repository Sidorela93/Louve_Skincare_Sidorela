To view and run the Louve Skincare project on your local machine using XAMPP, follow these steps:

Option 1: Clone via Git
Open your terminal (Command Prompt, Git Bash, or VS Code terminal).

Navigate to your htdocs directory in XAMPP:

cd C:/xampp/htdocs/
Clone the repository:

git clone https://github.com/Sidorela93/Louve_Skincare_Sidorela.git
Open XAMPP and start the Apache server.

Open your browser and visit:

http://localhost/Louve_Skincare_Sidorela
Option 2: Download as ZIP
Visit the repository:
https://github.com/Sidorela93/Louve_Skincare_Sidorela

Click the green Code button and choose Download ZIP.

Extract the ZIP file to the following path:

C:/xampp/htdocs/
Open XAMPP and start Apache.

Open your browser and go to:

http://localhost/Louve_Skincare_Sidorela
